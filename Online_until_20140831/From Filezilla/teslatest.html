Tesla Internship

Systems Testing

The Context

I spent May-August 2013 at Tesla Motors in Mountain View, California. At the time of my internship, Tesla's EV sedan Model S was launching. I was on the Systems Testing team, troubleshooting issues relating to user interaction with the Model S.

The Project

These vehicle behavior tests took time and had to be repeated for every new firmware version released on a car. We decided to automate a sizeable portion of the manual regression tests. I worked with a fellow intern to build the electromechanical and software system.

Result: Saved the team 2+ hours of manual testing per firmware version (released every few days).

The Design

The mechatronic system built for carrying out the user-facing tests on the Model S includes:

  • Actuator system that presses buttons on the 17" UI
  • Simulations that mimic vehicle states,
  • Script that runs tests (written in C).

The Reflection

My internship was an amazing experience. I went full-circle - from manually testing to building a mechatronic system to do the tests for me. I enjoyed the fast-paced environment at Tesla and I am grateful for the opportunity to work on a compelling, beautiful product.
